Инструкция по использованию:

• <iostream.h> — заголовочный файл для работы с вводом/выводом в C++.
• <cstdlib> — заголовочный файл для стандартных функций C (например, rand() и srand()).

⸻

Типы данных:

1. int
Целочисленный тип. Примеры: 1, 5, 999.

2. char
Символьный тип. Примеры: a, b, или строки вида "hello".

3. bool
Логический тип (true / false).
• Тип bool используется в условиях (if, while, for) и логических выражениях.
• По умолчанию значение bool — false.
• Заметьте: System.Boolean относится к C# / .NET и не связан с C++.

4. float
Тип для вещественных чисел с плавающей запятой. Примеры: 1.1, 5.4, 999.3.

5. double
Тип с высокой точностью для вещественных чисел.
• double более точен, чем float.
• При использовании double рекомендуется избегать умножения/деления с float, так как это может привести к потере точности.

⸻

Дополнительные примечания:
• Заголовок <iostream.h> устарел в C++. Используйте <iostream> вместо него.
• Тип bool не поддерживает значения, кроме true и false.
• float может содержать дробные числа, но его точность ограничена.
• При работе с float учитывайте, что арифметические операции могут привести к ошибкам из-за ограниченной точности.
